### Insurance Charges Prediction

This project uses a simple Linear Regression model to predict medical insurance charges based on features such as age, BMI, smoking status, and other personal details.

The dataset is first cleaned and preprocessed by removing duplicates and encoding categorical variables. A new feature (BMI category) is also created to improve model understanding.

The data is then split into training and testing sets, and a Linear Regression model is trained on the training data. The model is evaluated using R-squared and Adjusted R-squared metrics.

The final model achieves an R-squared score of around 79%, indicating a reasonable fit for this dataset.
